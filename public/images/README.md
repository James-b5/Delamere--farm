## Image Guidelines

### Recommended Sizes:
- **Hero images**: 1920x1080 or 1200x630
- **Product images**: 800x800 (square)
- **Blog images**: 1200x630
- **Thumbnails**: 400x400

### Formats:
- Use **JPG** for photos
- Use **PNG** for logos/icons
- Use **WebP** for better compression (optional)

### Naming Convention:
- Use lowercase
- Use hyphens (-) instead of spaces
- Be descriptive: `friesian-dairy-cow.jpg`

## How to Add Images:
1. Take photos of your farm, products, and livestock
2. Resize/crop to recommended dimensions
3. Save with descriptive names in the appropriate folder
4. The images will automatically appear on the website
